QSign API для text-to-sign планов и AI-video handoff

API отдаёт машинно-читаемый план, трассу покрытия, render-plan и экспортные пакеты для внешнего видео-процесса.

Ключевые endpoints

EndpointНазначение
POST /v1/translate/textСоздать sign-plan из текста.
GET /v1/sourcesПолучить реестр источников и статусы лицензий.
GET /v1/lexicon?language=ruПосмотреть публично доступные словарные записи.
GET /v1/jobs/{job_id}/render-planПолучить план рендеринга для сохранённой записи.
GET /v1/jobs/{job_id}/ai-video-briefПолучить universal prompt, operator handoff, render contract и JSON payload.
